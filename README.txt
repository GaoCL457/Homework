这次作业是我第一次写HTML代码，我从零学起。我大部分的HTML语言知识是从菜鸟教程上学来的，另外的部分是从助教老师的讲解和提供的网站上获取的。第一次写HTML代码的过程中确实遇到了很多问题。第一个问题出现在最一开始的编程软件的安装上，本想用WebStorm编写代码但是安装完后发现没有试用期，于是不得不选择VSCode，不过我觉得VSCode也挺好用的，配色上很舒服。代码中超链接的引用需要“href”，需要注意不能写成“herf”，要不然你找一个小时都找不出你错在哪。代码中的CSS引自Bootstrap，我对Bootstrap中栅格结构的分割方式进行了调整，导航栏部分我自行引入时只能做到普通的导航栏，无法通过利用其提供的代码将其修改为带下拉菜单的导航栏，因而我选择了复制粘贴。导航栏我大致分为左右两部分，左边分为七部分，分别是：网站名称、主页、第二大题的四道题目、第一大题（下拉菜单为四道小题），右边分为两部分：作者与关于。页面中的标题和正文的格式均为本人独立编写，另外“返回顶部”标签也为本人编写。计算机学科各课程之间的关系图我用亿图脑图软件绘制。参考资料中的所有网络链接和除WebStorm外的所有软件安装包均为本人查找，所有的PPT课件均来自本校老师分享。首页的图片“2021”由我在2021年观看过的我最喜欢的电影的海报组成，“2022”由我最想在2022年观看的电影的海报组成，这两张图片均由我在网络上搜集图片并通过PS合成。首页底部为我的GitHub主页地址和我的Gmail邮箱。另外第一大题虽然未让用HTML完成，但是既然第二大题用HTML完成了，为了网页的完整性，我将第一大题的内容也加入到网页中。
第一次写HTML代码，如有不当之处，还请多多包涵。
